# Story Image Generator (استوری جنریتور)

یک ابزار پیشرفته و کاملاً موبایل فرندلی برای ساخت تصاویر استوری حرفه‌ای با قابلیت‌های شخصی‌سازی کامل.

## ✨ ویژگی‌های کلیدی

### 📱 طراحی موبایل فرست
- **ریسپانسیو کامل**: عملکرد بهینه روی موبایل، تبلت و دسکتاپ
- **ناوبری پایینی**: دسترسی آسان به تمام بخش‌ها با لمس
- **Bottom Sheet**: پنل‌های کشویی با امکان سوایپ
- **Safe Area**: سازگاری با نات‌ها و گوشه‌های منحنی
- **Touch Friendly**: حداقل 44px برای تمام المنت‌های لمسی

### 🎨 تم‌های بصری حرفه‌ای
- **Night Radio**: تم تیره طلایی با فضای رادیویی
- **Aurora Glass**: شیشه‌ای با رنگ‌های شفق قطبی
- **Editorial Paper**: کاغذی ویرایشی برای محتوای رسمی
- **Luxury Noir**: لوکس مشکی با لمسات طلایی

### 📐 انواع لایوت
- **Meeting Layout**: برای آخرین ملاقات‌ها
- **Grid Layout**: برای آمار و اعداد
- **Ring Layout**: برای درصدها و پیشرفت
- **Timeline Layout**: برای نقاط عطف
- **Quote Layout**: برای نقل‌قول‌ها
- **Blank Layout**: برای طراحی آزاد

### 🔧 شخصی‌سازی کامل
- **6 رنگ تاکیدی**: انتخاب از پالت رنگی حرفه‌ای
- **تنظیم متن**: عنوان، زیرعنوان، مقدار، توضیح
- **چیدمان**: راست یا چپ‌چین
- **گردی گوشه‌ها**: 0 تا 60 پیکسل
- **شدت سایه**: 0 تا 100 درصد
- **واترمارک و برند**: قابل فعال/غیرفعال

### 📏 فرمت‌های خروجی
- **Instagram Story**: 1080×1920
- **Instagram Post**: 1080×1350  
- **Square**: 1080×1080
- **LinkedIn/X**: 1200×630

## 🚀 نحوه استفاده

### روی دسکتاپ
1. از سایدبار چپ دسته‌بندی مورد نظر را انتخاب کنید
2. از پنل راست قالب دلخواه را انتخاب کنید
3. در تب "شخصی‌سازی" تنظیمات را اعمال کنید
4. فرمت خروجی را از بالای صفحه انتخاب کنید
5. روی "دانلود PNG" کلیک کنید

### روی موبایل
1. از ناوبری پایینی روی "دسته‌بندی" بزنید
2. دسته مورد نظر را انتخاب کنید
3. از "گالری" قالب دلخواه را انتخاب کنید
4. از "تنظیمات" شخصی‌سازی کنید
5. از بالای صفحه فرمت را انتخاب و دانلود کنید

## 🎯 نکات تکنیکی

### Responsive Breakpoints
```css
/* موبایل */ 
@media (max-width: 599px)

/* تبلت */
@media (min-width: 600px) and (max-width: 980px)  

/* دسکتاپ */
@media (min-width: 981px)
```

### Touch Gestures
- **سوایپ پایین**: بستن Bottom Sheet
- **تپ روی Handle**: بستن Bottom Sheet
- **زوم**: دکمه‌های + و - کنار پیش‌نمایش

### Canvas Rendering
- **رزولوشن بالا**: رندر در رزولوشن کامل
- **بهینه‌سازی**: استفاده از Canvas API برای کیفیت بهتر
- **فونت‌ها**: Fraunces، Inter، Vazirmatn
