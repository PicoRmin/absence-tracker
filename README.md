# Storycraft — Story Image Generator

یک پروژه تولید استوری اینستاگرام وب‌محور با تمرکز بر:
- طراحی زیبا و موبایل فرندلی
- رندر پویا با HTML5 Canvas
- شخصی‌سازی انواع قالب و تم
- ساختار چندفایلی برای نگهداری بهتر

## 🚀 وضعیت فعلی پروژه
نسخه فعلی پروژه به صورت چندفایلی بازسازی شده است و دیگر یک فایل HTML بزرگ نیست.

### فایل‌های اصلی
- `index.html`  : ساختار صفحه و placeholderهای رابط کاربری
- `styles.css` : تمام استایل‌ها و ریسپانسیو
- `data.js`    : داده‌های قالب، تم‌ها و دسته‌بندی‌ها
- `utils.js`   : توابع مشترک رندر و کمک‌کننده
- `app.js`     : مدیریت state، ساخت گالری، رویدادها و رندر Canvas
- `roadmap.md` : نقشه راه توسعه پروژه

## 🎯 ویژگی‌های مهم
- طراحی ریسپانسیو برای موبایل، تبلت و دسکتاپ
- گالری قالب‌های قابل انتخاب و پیش‌نمایش
- پنل شخصی‌سازی با تم، رنگ تاکیدی، متن و تنظیمات قالب
- زوم پیش‌نمایش و انتخاب فرمت خروجی
- دانلود خروجی PNG
- ذخیره‌سازی تنظیمات با `localStorage`

## 🧩 نحوه اجرا
برای مشاهده پروژه کافی است `index.html` را در مرورگر باز کنید.

اگر می‌خواهید با سرور محلی اجرا کنید:
```bash
cd /workspaces/absence-tracker
python3 -m http.server 8000
```
سپس به `http://127.0.0.1:8000` بروید.

## 📌 نکات مهم
- فایل `story-image-generator.html` نسخه اولیه تک‌فایلی پروژه است.
- نسخه فعلی از آن جدا شده و منطق پروژه اکنون در `app.js` و `utils.js` قرار دارد.
- `roadmap.md` برنامه توسعه بعدی را به زبان فارسی توضیح داده است.

## 🔧 ساختار جدید
- UI در `index.html`
- استایل‌ها در `styles.css`
- داده‌های تم و قالب در `data.js`
- منطق رندر و کمکی در `utils.js`
- کنترل state و event handling در `app.js`
