Professional Product Design Prompt
Project Overview
Design a premium mobile application called Memory Timeline / Absence Tracker.
This is not a simple countdown timer.
It is an emotional timeline that combines the concepts of a time tracker, personal journal, calendar, life dashboard, memory archive, and emotional analytics into one elegant experience.
The application’s core philosophy is:
“Time should not only be counted—it should be experienced.”
Every time the user opens the app, they should feel the passage of time from multiple perspectives: chronological, emotional, seasonal, astronomical, and personal.
The design language should be calm, minimal, cinematic, elegant, and emotionally immersive without becoming melodramatic.
________________________________________
Design Principles
•	Minimal UI
•	Premium Apple-level aesthetics
•	Smooth animations (60 FPS)
•	Soft gradients
•	Elegant typography
•	Emotional storytelling through data
•	Dark mode first
•	Glassmorphism used subtly
•	Highly polished micro-interactions
•	Beautiful empty states
•	No visual clutter
The experience should feel closer to an interactive memory museum than a productivity app.

Overall Experience Goal
The application should not feel like a stopwatch.
It should feel like opening a beautifully designed museum of memories, where every statistic tells a story and every passing day gains emotional meaning.
The product should evoke quiet reflection rather than sadness, transforming the passage of time into an elegant, deeply personal narrative through thoughtful design, meaningful data visualization, and immersive interactions.
